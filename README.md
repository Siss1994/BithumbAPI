# 🎮 블록커스 (Blokus) 웹 게임

블록커스 보드게임을 웹으로 구현한 멀티플레이어 게임입니다.

## 🎯 게임 소개

블록커스는 전략적 보드게임으로, 플레이어들이 20x20 보드에 폴리오미노 조각들을 배치하는 게임입니다.

### 게임 규칙

1. **시작**: 각 플레이어는 21개의 폴리오미노 조각을 가지고 시작합니다
2. **첫 수**: 자신의 시작 코너(★)에 조각을 배치해야 합니다
3. **배치 규칙**:
   - 같은 색 조각끼리는 모서리(대각선)로만 닿아야 합니다
   - 같은 색 조각끼리 변이 닿으면 안됩니다
   - 다른 색 조각과는 어떻게 닿아도 상관없습니다
4. **승리 조건**: 모든 플레이어가 더 이상 놓을 수 없을 때 게임이 종료되며, 남은 조각이 가장 적은 플레이어가 승리합니다

### 점수 계산

- 사용하지 못한 조각 1칸당 -1점
- 모든 조각을 다 놓은 경우 +15점
- 마지막 조각이 1칸짜리인 경우 추가 +5점

## 🕹️ 게임 모드

### 1 vs 1 (2인)
- 2명이 대결하는 모드
- 플레이어 1 (파란색) vs 플레이어 2 (빨간색)

### 1v1v1v1 (4인)
- 4명이 개인전으로 대결하는 모드
- 각 플레이어가 코너에서 시작

### 2 vs 2 (팀전)
- 4명이 2개 팀으로 나뉘어 대결하는 모드
- 팀 1: 플레이어 1, 2
- 팀 2: 플레이어 3, 4

## 🎮 조작 방법

### 마우스
- **조각 선택**: 왼쪽 패널에서 조각 클릭
- **조각 배치**: 보드에서 원하는 위치 클릭

### 버튼
- **↻ 회전**: 선택한 조각을 시계방향으로 90도 회전
- **⇄ 좌우반전**: 선택한 조각을 좌우로 반전
- **⇅ 상하반전**: 선택한 조각을 상하로 반전
- **패스**: 현재 턴을 넘김

### 키보드 단축키
- `R`: 회전
- `H`: 좌우반전
- `V`: 상하반전
- `P`: 패스
